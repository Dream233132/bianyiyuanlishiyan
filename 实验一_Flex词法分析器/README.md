# 实验一：Flex词法分析器

## 实验目的
使用Flex工具自动构造C--语言的词法分析器，识别C--语言的所有单词符号并输出二元组形式。

## C--语言单词符号

### 1. 关键字（种别码1-5）
| 关键字 | 种别码 |
|--------|--------|
| while  | 1      |
| if     | 2      |
| else   | 3      |
| switch | 4      |
| case   | 5      |

### 2. 标识符和常数
| 类型   | 种别码 | 说明 |
|--------|--------|------|
| 标识符 | 6      | 字母或下划线开头，后跟字母、数字或下划线 |
| 常数   | 7      | 整数、浮点数、科学计数法 |

### 3. 运算符
| 运算符 | 种别码 | 助记符 | 说明 |
|--------|--------|--------|------|
| +      | 8      | +      | 加法 |
| ?      | 9      | ?      | 问号 |
| *      | 10     | *      | 乘法 |
| <=     | 10     | LE     | 小于等于 |
| <      | 11     | LT     | 小于 |
| ==     | 12     | EQ     | 等于 |
| =      | 13     | =      | 赋值 |
| :      | 13     | :      | 冒号 |
| **-**  | **14** | **-**  | **减法（新增）** |
| **/**  | **15** | **/**  | **除法（新增）** |
| **>**  | **16** | **GT** | **大于（新增）** |
| **>=** | **17** | **GE** | **大于等于（新增）** |

### 4. 特殊符号
| 符号 | 种别码 | 说明 |
|------|--------|------|
| (    | 18     | 左括号 |
| )    | 19     | 右括号 |
| {    | 20     | 左花括号 |
| }    | 21     | 右花括号 |
| ;    | 22     | 分号 |
| ,    | 23     | 逗号 |

## 文件说明

- **lexer.l**: Flex源文件，定义词法规则
- **test_correct.c**: 正确的C--测试程序
- **test_error.c**: 含有词法错误的测试程序
- **Makefile**: 编译和测试脚本

## 编译和运行

### Windows环境（需要安装Flex和GCC）

```bash
# 编译词法分析器
flex lexer.l
gcc lex.yy.c -o lexer.exe

# 测试正确的程序
./lexer test_correct.c

# 测试错误的程序
./lexer test_error.c
```

## 输出格式

词法分析器输出二元组格式：`(种别码, 值)`

示例：
```
(2, if)          // if关键字
(18, ()          // 左括号
(6, x)           // 标识符x
(16, GT)         // 大于运算符
(7, 10)          // 数字10
(19, ))          // 右括号
```

## 词法错误处理

当遇到非法字符时，输出错误信息：
```
词法错误：第13行出现非法字符 '@'
```

## 实现要点

1. **正则表达式定义**：使用Flex的正则表达式定义各类单词符号
2. **优先级处理**：双字符运算符（<=, >=, ==）必须在单字符运算符之前匹配
3. **数字识别**：支持整数、浮点数和科学计数法（如2.71828e10）
4. **注释处理**：支持单行注释（//）和多行注释（/* */）
5. **错误检测**：识别并报告非法字符

## 测试用例说明

### test_correct.c
- 包含所有关键字、运算符的正确使用
- 测试加减乘除四则运算
- 测试所有比较运算符（<, <=, >, >=, ==）
- 测试浮点数和科学计数法

### test_error.c
- 包含多种非法字符（@, #, $, &, %, ^, ~, `, |）
- 用于测试词法分析器的错误检测能力
- **注意**：此文件包含故意的词法错误，C编译器会报错，这是正常的。该文件仅用作词法分析器的输入测试，不需要通过C编译器编译。
